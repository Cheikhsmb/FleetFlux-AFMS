# FleetFlux - HTML Presentation

## How to Use

1. **Open the presentation**: Simply open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari)

2. **Navigation**:
   - Click the **Next** and **Previous** buttons in the bottom-right corner
   - Use **Arrow Keys** (Left/Right) to navigate
   - Press **Spacebar** to go to next slide

3. **Features**:
   - Progress bar at the top shows current position
   - Slide counter in top-right corner
   - Smooth animations and transitions
   - Beautiful color scheme matching Logidoo brand (Orange)
   - Code syntax highlighting
   - Responsive design

## Customization

- **Your Name & Course**: Edit Slide 1 in `index.html`
- **Logo**: The logo path is already set to `../src/com/logidoo/services/LogidoLogo.png`
- **Colors**: Brand color is set to orange (#FF8C00) - you can change it in the Tailwind config section

## Technical Details

- Built with **Tailwind CSS** (via CDN)
- Pure HTML/CSS/JavaScript - no build process needed
- Font: Inter (Google Fonts)
- Fully responsive
