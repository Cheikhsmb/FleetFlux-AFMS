5 OOP: Encapsulation

Vehicle.java
public abstract class Vehicle {
    // Private fields - hidden from outside
    private double mileage;
    private double fuelLevel;
    
    // Controlled access via getters
    public double getMileage() {
        return mileage;
    }
    
    // Validation in setters
    public void 
        setMileage(double mileage) {
        if (mileage >= 0) {
            this.mileage = mileage;
        }
    }
}

Key Points

  • üîí
    All fields are private - cannot be accessed directly
  • üîë
    Access through controlled methods (getters/setters)
  • ‚úì
    Validation in setters (mileage >= 0, fuel 0-100%)

Why: Data protection, prevents invalid values, controlled access
